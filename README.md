# food-order-app
리액트를 사용해서 만든 음식 주문 배달 웹 어플리케이션입니다.

## 기술 스택  
- 리액트로 컴포넌트 생성
- css모듈로 스타일링
- 컨텍스트 API를 활용하여 장바구니 상태 관리
- 파이어베이스 연동하여 장바구니 상태 저장 및 불러오기
